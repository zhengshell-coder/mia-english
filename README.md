# Mia · 随时英语老师 🎓

一个孜孜不倦、随时陪你聊天的 AI 英语老师。纯网页 PWA,装到手机主屏幕像原生 App 一样用,**没有时间限制**,调用 Gemini 免费 API,数据全部保存在你自己手机上。

## 功能

- 🎤 **语音对话**:点一下麦克风直接说英语,Mia 听懂后用语音回复(也可以打字)
- 🔁 **免提连聊**:她说完自动开始听你说,你说完停顿一下自动发送,像打电话一样
- 🇨🇳 **中文随时提问**:聊天中可以用中文问"这个用英语怎么说 / 这个语法什么意思",她解释完再带你回到英语对话
- ✏️ **温和纠错**:你说错的句子,她会在气泡下方给出更地道的说法和中文说明
- 📒 **学习笔记**:页面下半部分自动记录对话中出现的关键词汇、短语、语法,点开看中文释义和例句
- 📅 **每日记录**:每天聊了什么自动总结归档,词汇笔记按天保存,还有连续学习天数 🔥
- 🧠 **自适应**:根据你的水平、弱点和兴趣自动调整话题和难度,并在后续对话中"复现"之前学过的词

## 三步用起来

### 第 1 步:申请免费的 Gemini API Key

1. 打开 <https://aistudio.google.com/apikey>
2. 用 Google 账号登录,点 **Create API key**
3. 复制生成的 Key(免费,无需绑卡)

### 第 2 步:发布到 GitHub Pages

不需要安装任何工具,网页上点几下即可:

1. 登录 GitHub,点右上角 **+** → **New repository**,名字随意(例如 `mia-english`),选 **Public**,创建
2. 在仓库页面点 **Add file → Upload files**,把本文件夹里的所有文件拖进去,点 **Commit changes**
3. 进入仓库 **Settings → Pages**,Source 选 **Deploy from a branch**,Branch 选 **main** + **/(root)**,保存
4. 等一两分钟,页面顶部会显示你的网址,形如:
   `https://你的用户名.github.io/mia-english/`

### 第 3 步:装到手机上

用手机浏览器打开上面的网址:

- **iPhone(Safari)**:点底部「分享」按钮 → **添加到主屏幕**
- **安卓(Chrome)**:点右上角菜单 → **添加到主屏幕 / 安装应用**

打开 App,粘贴你的 API Key,就可以开始聊了。

## 免费额度说明

Gemini 免费档按模型限速(2026 年中,数值可能调整):

| 模型 | 每分钟 | 每天 |
|---|---|---|
| gemini-3.5-flash(默认) | ~10 句 | 250~1500 句 |
| gemini-flash-lite-latest | ~15 句 | 最多(推荐备用) |

正常聊天完全够用。如果提示限速,等几十秒,或在 **设置 → 模型** 里换 `gemini-flash-lite-latest`。

## 隐私

- API Key 和所有聊天记录、笔记**只保存在你手机浏览器本地**(localStorage),不经过任何第三方服务器
- 语音和文字只发给 Google Gemini API 用于生成回复
- 设置里可以随时**导出/导入**数据(换手机时用),也可以一键清空
- 注意:不要把你的 API Key 写进代码上传到 GitHub,App 会在首次打开时让你在手机上输入

## 常见问题

**点麦克风没反应?**
浏览器会先弹出麦克风权限请求,要选「允许」。iPhone 上必须用 Safari 打开/安装。

**Mia 不出声?**
先随便点一下屏幕再试(手机浏览器要求首次播放语音前有触摸操作);确认没开静音键,设置里「自动朗读」是开的。

**提示限速/429?**
免费额度到了每分钟上限,稍等再说,或换 flash-lite 模型。

**想换口音/语速?**
语速在设置里调;口音取决于手机系统自带的英语语音(iPhone:设置→辅助功能→朗读内容→声音 里可下载更自然的声音)。

**想调整 Mia 的教学风格?**
打开 `app.js`,搜索 `buildSystem`,里面那段英文就是她的"人设说明书",随意修改后重新上传即可。
